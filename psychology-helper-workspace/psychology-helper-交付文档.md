### 触发描述

```
心理学难题解决助手 - 基于 20 个经典心理学实验，帮助普通人理解和解决日常生活中的心理困惑。当用户提到情绪问题、人际关系、职场压力、家庭矛盾、自我认知、行为习惯等心理相关话题时，使用此技能提供专业且实用的心理学建议。涵盖：自信、拖延、焦虑、恐惧、依恋、服从、从众、决策、良知、正念等主题。
```

```

```
