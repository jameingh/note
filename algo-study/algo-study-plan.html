第一阶段
基础认知：概念与复杂度
建立对数据结构和算法的整体认知，理解时间 / 空间复杂度的本质，这是一切后续学习的地基。
建议 1 周
💡
学习建议：先通读不求甚解，重点在理解"什么是数据结构"和"为什么需要算法分析"。复杂度只需要记住 Big-O 的概念，能看懂 O(n)、O(n²) 即可进入下一阶段。
Lesson 01
✓
什么是数据结构？
  • 数据结构的定义：数据元素集合 + 关系 + 操作方法
  • Python 的 list/set/dict 都是高级数据结构
  • 逻辑结构 vs 物理结构的区别
⏱ 20 分钟 基础
Lesson 02
✓
数据结构分类
  • 逻辑结构：线性（数组/链表/栈/队列）vs 非线性（树/图）
  • 物理结构：连续存储（数组）vs 分散存储（链表）
  • 一切数据结构都是数组+链表的组合
⏱ 20 分钟 基础
Lesson 03
✓
算法基础与分类
  • 算法的 5 大特性：输入、输出、有穷性、确定性、可行性
  • 按目的分：搜索 / 排序 / 最优化算法
  • 按策略分：暴力法、分治、动态规划、贪心
⏱ 20 分钟 基础
Lesson 04
✓
时间复杂度 Big-O
  • T(n) 的概念：基本指令数 vs 绝对运行时间
  • 最优 / 最坏 / 平均，重点关注最坏情况
  • O(1) < O(logn) < O(n) < O(nlogn) < O(n²)
  • 典型案例：两数求和、二分查找、冒泡排序
⏱ 40 分钟 重点
Lesson 05
✓
空间复杂度
  • 内存空间随输入规模增大的变化趋势
  • 时间换空间 vs 空间换时间的权衡
  • 常见空间复杂度案例分析
⏱ 30 分钟 重点
🔗
第二阶段
线性数据结构
掌握数组、链表、栈、队列、哈希表这 5 种线性结构，理解它们的实现原理和操作复杂度。
建议 2 周
💡
学习建议：每个结构都要自己动手实现一遍（Python），关注每个操作的时间复杂度。理解之后配合 LeetCode 练习 2~3 道相关题目，巩固印象。
Lesson 06
✓
数组：原理与实现
  • 连续内存 + 随机访问 O(1)
  • 创建、扩容（双倍扩容策略）
  • 插入 O(n) / 删除 O(n) / 查找 O(n) / 访问 O(1)
  • 完整代码实现
⏱ 45 分钟 基础
Lesson 07
✓
链表：原理与实现
  • 节点 + 指针，分散存储
  • 插入/删除 O(1)（已知节点），访问 O(n)
  • 单链表完整实现：增删改查遍历
  • 与数组的对比：各自适用场景
⏱ 50 分钟 重点
Lesson 08
✓
栈：LIFO 结构
  • 后进先出（LIFO）原则
  • push / pop / peek 操作
  • 经典应用：括号匹配、函数调用栈、表达式求值
  • 用数组 or 链表实现
⏱ 40 分钟 基础
Lesson 09
✓
队列：FIFO 结构
  • 先进先出（FIFO）原则
  • enqueue / dequeue 操作
  • 应用场景：BFS、任务调度
  • 循环队列优化
⏱ 35 分钟 基础
Lesson 10
✓
哈希表：O(1) 的魔法
  • 哈希函数的作用与设计
  • 哈希冲突：链式法 vs 开放寻址
  • 平均 O(1) 的插入/查询/删除
  • 完整实现 + 负载因子与扩容
⏱ 60 分钟 重点
🌳
第三阶段
非线性结构：树与图
树和图是更复杂的数据结构，也是面试高频考点。重点掌握二叉搜索树和图的遍历算法。
建议 2 周
💡
学习建议：树的遍历（前序/中序/后序/层序）要烂熟于心。图的 DFS/BFS 遍历先理解思想，再看代码，这两个遍历是所有图算法的基础。
Lesson 11
✓
树的基本概念
  • 树的术语：根、叶、高度、深度、度
  • 二叉树：每个节点最多 2 个子节点
  • 存储结构：链式表示 vs 数组表示
  • 常见特殊二叉树：满二叉树、完全二叉树
⏱ 35 分钟 基础
Lesson 12
✓
二叉搜索树（BST）操作
  • BST 性质：左 < 根 < 右
  • 查找：O(logn) 平均，O(n) 最坏
  • 插入 + 删除（三种情况）
  • 完整 Python 实现
⏱ 60 分钟 进阶
Lesson 13
✓
二叉树的 4 种遍历
  • 前序遍历：根 → 左 → 右
  • 中序遍历：左 → 根 → 右（BST 输出有序）
  • 后序遍历：左 → 右 → 根
  • 层序遍历（BFS）：按层从上到下
⏱ 50 分钟 重点
Lesson 14
✓
图的概念与表示
  • 有向图 vs 无向图，带权图 vs 无权图
  • 邻接矩阵：直观，空间 O(V²)
  • 邻接表：节省空间，常用表示
  • 度（入度/出度）的概念
⏱ 40 分钟 重点
Lesson 15
✓
图的遍历：DFS & BFS
  • 深度优先搜索（DFS）：栈/递归，探到底再回溯
  • 广度优先搜索（BFS）：队列，一层层扩散
  • 已访问标记防止死循环
  • 适用场景对比
⏱ 55 分钟 进阶

第四阶段
基础算法：查找与排序
掌握最常用的查找和排序算法，从简单到复杂，理解每种算法的思路、复杂度和适用场景。
建议 2 周
💡
学习建议：排序算法不要死记代码，而是理解每种算法的核心思想。归并排序和快速排序的分治思想非常重要，会在后续算法阶段反复用到。
🎯 LeetCode 推荐：LC-704（二分）/ LC-169（多数元素）
Lesson 16
✓
二分查找
  • 前提：有序数组
  • 每次缩减一半，O(logn)
  • 边界处理：left ≤ right vs left < right
  • LeetCode 704：标准二分
⏱ 40 分钟 基础
Lesson 17
✓
查找多数元素（Boyer-Moore）
  • 摩尔投票算法：O(n) 时间，O(1) 空间
  • 出现次数 > n/2 的元素一定存在
  • LeetCode 169：多数元素
⏱ 35 分钟 重点
Lesson 18
✓
O(n²) 排序三兄弟
  • 冒泡排序：相邻比较，最简单
  • 选择排序：每轮找最小，交换到前面
  • 插入排序：像打牌，插入有序区
  • 三者对比：稳定性 / 最好最坏情况
⏱ 50 分钟 基础
Lesson 19
✓
归并排序 O(nlogn)
  • 分治思想：分割 → 递归排序 → 合并
  • 稳定排序，O(n) 额外空间
  • 理解递归树，每层 O(n) 总共 O(logn) 层
⏱ 50 分钟 重点
Lesson 20
✓
快速排序 + 堆排序
  • 快排：选 pivot，分区，递归；平均 O(nlogn)
  • 最坏 O(n²)，随机化可优化
  • 堆排序：构建最大堆 + 反复提取根
  • 堆排序 O(nlogn) 且不需要额外空间
⏱ 60 分钟 进阶

第五阶段
高阶算法：分治 · DP · 回溯 · 贪心
算法设计的四大范式，是解决复杂问题的核心工具。结合 LeetCode 经典题目理解每种范式的应用场景。
建议 3 周
💡
学习建议：动态规划是最难的部分，不要急，先从"爬楼梯"这类简单 DP 入手，重点理解"状态定义"和"状态转移方程"。回溯算法记住三步：选择 → 递归 → 撤销。
🎯 LeetCode 推荐：LC-70（爬楼梯）/ LC-53（最大子数组）/ LC-46（全排列）
Lesson 21
✓
分治算法
  • 三步：分解 → 递归求解 → 合并
  • 案例一：汉诺塔（经典递归）
  • 案例二：Karatsuba 大整数乘法
  • 适用条件：子问题相互独立
⏱ 50 分钟 重点
Lesson 22
✓
动态规划入门
  • 核心：最优子结构 + 重叠子问题
  • DP 三步：定义状态 → 状态转移方程 → 初始值
  • 案例：爬楼梯 f(n) = f(n-1) + f(n-2)
  • LeetCode 70
⏱ 60 分钟 重点
Lesson 23
✓
动态规划进阶：背包问题
  • 最大子数组之和：f(i) = max(f(i-1)+nums[i], nums[i])
  • 0-1 背包：每件物品只能选 0 次或 1 次
  • 完全背包：物品可无限选取
  • 一维数组空间优化技巧
⏱ 90 分钟 进阶
