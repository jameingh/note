# 情感联结技能实现计划

## 🎯 项目目标
创建一个专业的情感联结技能，帮助用户与他人建立和维护深层情感连接，提升人际关系质量。

## 📋 技能结构

### 1. 技能基本信息
- **技能名称**：emotional-connection-builder
- **技能描述**：情感联结构建者 - 帮助用户与亲人、朋友、伴侣等建立深层情感连接，提升沟通质量和关系亲密度。
- **目标用户**：希望改善人际关系的普通用户
- **应用场景**：家庭关系、亲密关系、友谊维护等

### 2. 核心功能模块

#### 模块一：情感联结基础认知
- 什么是情感联结
- 情感联结的重要性
- 常见障碍和挑战

#### 模块二：核心技能方法
- 深度倾听技巧
- 情绪验证方法
- 脆弱分享策略
- 日常连接建立
- 非暴力沟通
- 共同体验创造

#### 模块三：场景应用
- 亲子关系
- 夫妻/伴侣关系
- 代际关系（与父母）
- 友谊维护

#### 模块四：实践系统
- 日常练习指南
- 场景演练脚本
- 反思工具
- 进步追踪

#### 模块五：资源库
- 对话示例库
- 练习模板
- 常见问题解答

## 🛠️ 技术实现

### 1. 技能文件结构
```
skills/
└── emotional-connection-builder/
    ├── SKILL.md          # 技能主文件
    ├── references/       # 参考资料
    │   ├── dialogues.md  # 对话示例
    │   └── exercises.md  # 练习模板
    └── evals/            # 评估文件
        └── evals.json    # 评估配置
```

### 2. 实现步骤

1. **创建技能目录结构**
   - 创建 `skills/emotional-connection-builder/` 目录
   - 创建必要的子目录和文件

2. **编写 SKILL.md 主文件**
   - 技能基本信息
   - 核心功能模块
   - 处理逻辑
   - 输出格式
   - 对话示例

3. **编写参考资料**
   - 对话示例库（正反例子）
   - 练习模板

4. **配置评估文件**
   - 定义评估标准
   - 设置测试用例

### 3. 技能核心逻辑

#### 处理流程
1. **问题识别**：分析用户的具体情感联结问题
2. **情境分析**：理解问题发生的具体场景
3. **方法匹配**：选择合适的情感联结技巧
4. **提供方案**：给出具体可操作的建议
5. **实践指导**：提供练习方法和追踪工具

4. **温暖支持**：注重情感共鸣和鼓励
