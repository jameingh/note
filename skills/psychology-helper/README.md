## 🚀 如何使用

### 方式 1：直接使用（推荐）

在对话中直接提问心理学相关问题，技能会自动触发：

**示例问题**：
- "我总是拖延，怎么办？"
- "我很自卑，怎么建立自信？"
- "工作压力大，没有动力"
- "我很孤独，但没有精力社交"
- "在感情里总是没有安全感"
- "我很焦虑，静不下来"

### 方式 2：明确指定技能

```
请用 psychology-helper 技能帮我分析这个问题：
[你的问题]
```

## 🛠️ 技术信息

### 触发机制

技能在以下场景自动触发：
- 用户提到情绪问题（焦虑、抑郁、恐惧等）
- 用户提到人际关系困扰（亲密关系、职场关系等）
- 用户提到自我认知问题（自信、自尊、标签等）
- 用户提到行为习惯问题（拖延、冲动、成瘾等）
- 用户明确询问心理学相关问题

### 技能描述（用于触发）

```
心理学难题解决助手 - 基于 20 个经典心理学实验，帮助普通人理解和解决日常生活中的心理困惑。当用户提到情绪问题、人际关系、职场压力、家庭矛盾、自我认知、行为习惯等心理相关话题时，使用此技能提供专业且实用的心理学建议。涵盖：自信、拖延、焦虑、恐惧、依恋、服从、从众、决策、良知、正念等主题。
```
